Tout ce que vous devez savoir sur nos services de création de portfolio et de landing page

Notre offre à 250€ comprend la création complète d'un portfolio professionnel et personnalisé incluant :

  • Design sur mesure adapté à votre profil professionnel
  • Site entièrement responsive (adapté à tous les appareils)
  • Jusqu'à 5 sections personnalisées
  • Formulaire de contact fonctionnel
  • Optimisation SEO de base
  • Hébergement gratuit sur GitHub Pages
  • 2 révisions pour ajustements
  • Livraison en 7 jours

C'est une solution clé en main, sans frais cachés ni coûts supplémentaires.

Hébergement GitHub (inclus gratuitement) : Votre site sera accessible via une adresse de type username.github.io/votre-portfolio. C'est une solution fiable et gratuite, parfaite pour démarrer.

Option domaine personnalisé (+3€/mois) : Vous aurez une adresse web professionnelle de type www.votrenom.com ou www.votreportfolio.com. Cela renforce votre identité professionnelle et crédibilité. Cette option inclut également :

  • Certificat SSL (https://) pour la sécurité
  • Gestion du domaine
  • Support technique pour le domaine

Le choix dépend principalement de votre besoin de personnalisation et de l'image professionnelle que vous souhaitez projeter.

  1. Consultation initiale : Nous discutons de vos besoins, préférences et objectifs pour votre portfolio.
  2. Collecte des informations : Vous nous fournissez vos contenus (textes, images, projets à mettre en avant).
  3. Conception : Nous créons une première version de votre portfolio en 3-4 jours.
  4. Révision : Vous examinez le site et demandez jusqu'à 2 séries de modifications.
  5. Finalisation : Nous intégrons vos retours et peaufinons le site.
  6. Livraison : Votre portfolio est mis en ligne et vous recevez les accès pour le gérer si nécessaire.

Le processus complet prend généralement 7 jours, mais peut être accéléré en cas d'urgence.

Oui, vous avez deux options pour les mises à jour :

  1. Option simple : Nous pouvons configurer votre site pour qu'il soit facilement modifiable via une interface basique, même sans compétences techniques.
  2. Option avancée : Si vous avez des connaissances en développement web, nous vous donnerons accès au code source et au dépôt GitHub pour que vous puissiez faire des modifications plus poussées.

Nous proposons également un service de mise à jour ponctuelle au tarif de 30€ par intervention si vous préférez nous confier cette tâche.

Nos modalités de paiement sont simples et sécurisées :

  • Acompte : 50% (125€) à la commande pour démarrer le projet
  • Solde : 50% restant à la livraison, après validation de votre satisfaction

Nous acceptons les paiements par virement bancaire, PayPal ou carte bancaire.

Pour l'option domaine personnalisé, le coût de 3€/mois est facturé trimestriellement (9€) ou annuellement (36€ avec un mois offert).

Votre satisfaction est notre priorité :

  • Vous avez droit à 2 séries de révisions complètes incluses dans le prix
  • Si après ces révisions, vous n'êtes toujours pas satisfait, nous travaillerons avec vous pour résoudre les problèmes
  • Notre objectif est toujours d'atteindre 100% de satisfaction client

Dans le cas très rare où nous ne pourrions pas vous satisfaire malgré tous nos efforts, nous proposons un remboursement partiel, en fonction du travail déjà effectué.

Nos services s'adressent à une large gamme de professionnels, notamment :

  • Créatifs : Photographes, designers, illustrateurs, artistes
  • Développeurs : Programmeurs, intégrateurs, experts en informatique
  • Consultants : Formateurs, coaches, experts métier
  • Freelances : Rédacteurs, traducteurs, marketeurs
  • Entrepreneurs : Startups, indépendants, auto-entrepreneurs
  • Étudiants : Finissant leurs études et cherchant un emploi

Nous adaptons chaque portfolio au secteur d'activité et aux besoins spécifiques du client.

Ces deux types de sites répondent à des besoins différents :

Portfolio (250€)

  • Présente votre profil professionnel complet
  • Showcase de vos travaux et projets
  • Multiple sections (about, services, projets, contact)
  • Objectif : présenter votre parcours et compétences
  • Idéal pour : montrer votre expertise globale

Landing Page (180€)

  • Page unique avec un objectif précis
  • Focus sur un service/produit spécifique
  • Conception orientée conversion
  • Objectif : générer des leads ou des ventes
  • Idéal pour : promouvoir une offre particulière

Certains clients choisissent de combiner les deux : un portfolio complet avec des landing pages spécifiques pour certaines offres.

Tout ce que vous devez savoir sur nos services de création de portfolio et de landing page

Notre offre à 250€ comprend la création complète d'un portfolio professionnel et personnalisé incluant :

  • Design sur mesure adapté à votre profil professionnel
  • Site entièrement responsive (adapté à tous les appareils)
  • Jusqu'à 5 sections personnalisées
  • Formulaire de contact fonctionnel
  • Optimisation SEO de base
  • Hébergement gratuit sur GitHub Pages
  • 2 révisions pour ajustements
  • Livraison en 7 jours

C'est une solution clé en main, sans frais cachés ni coûts supplémentaires.

Hébergement GitHub (inclus gratuitement) : Votre site sera accessible via une adresse de type username.github.io/votre-portfolio. C'est une solution fiable et gratuite, parfaite pour démarrer.

Option domaine personnalisé (+3€/mois) : Vous aurez une adresse web professionnelle de type www.votrenom.com ou www.votreportfolio.com. Cela renforce votre identité professionnelle et crédibilité. Cette option inclut également :

  • Certificat SSL (https://) pour la sécurité
  • Gestion du domaine
  • Support technique pour le domaine

Le choix dépend principalement de votre besoin de personnalisation et de l'image professionnelle que vous souhaitez projeter.

  1. Consultation initiale : Nous discutons de vos besoins, préférences et objectifs pour votre portfolio.
  2. Collecte des informations : Vous nous fournissez vos contenus (textes, images, projets à mettre en avant).
  3. Conception : Nous créons une première version de votre portfolio en 3-4 jours.
  4. Révision : Vous examinez le site et demandez jusqu'à 2 séries de modifications.
  5. Finalisation : Nous intégrons vos retours et peaufinons le site.
  6. Livraison : Votre portfolio est mis en ligne et vous recevez les accès pour le gérer si nécessaire.

Le processus complet prend généralement 7 jours, mais peut être accéléré en cas d'urgence.

Oui, vous avez deux options pour les mises à jour :

  1. Option simple : Nous pouvons configurer votre site pour qu'il soit facilement modifiable via une interface basique, même sans compétences techniques.
  2. Option avancée : Si vous avez des connaissances en développement web, nous vous donnerons accès au code source et au dépôt GitHub pour que vous puissiez faire des modifications plus poussées.

Nous proposons également un service de mise à jour ponctuelle au tarif de 30€ par intervention si vous préférez nous confier cette tâche.

Nos modalités de paiement sont simples et sécurisées :

  • Acompte : 50% (125€) à la commande pour démarrer le projet
  • Solde : 50% restant à la livraison, après validation de votre satisfaction

Nous acceptons les paiements par virement bancaire, PayPal ou carte bancaire.

Pour l'option domaine personnalisé, le coût de 3€/mois est facturé trimestriellement (9€) ou annuellement (36€ avec un mois offert).

Votre satisfaction est notre priorité :

  • Vous avez droit à 2 séries de révisions complètes incluses dans le prix
  • Si après ces révisions, vous n'êtes toujours pas satisfait, nous travaillerons avec vous pour résoudre les problèmes
  • Notre objectif est toujours d'atteindre 100% de satisfaction client

Dans le cas très rare où nous ne pourrions pas vous satisfaire malgré tous nos efforts, nous proposons un remboursement partiel, en fonction du travail déjà effectué.
